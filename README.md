## :package: Installation

### **Cloning the repository**

```sh
  $ git clone git@github.com:VictorBDias/mindset-tkc.git
```

### **Running**

```sh
  $ cd mindset-tkc
  # Dependencies install.
  $ yarn
  # Runs the app in the development mode.
  $ yarn start
```
